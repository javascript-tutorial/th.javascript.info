# ฟังก์ชันลูกศร (Arrow functions) ข้อมูลพื้นฐาน

มีไวยากรณ์ที่เรียบง่ายและกระชับอีกแบบหนึ่งในการสร้างฟังก์ชัน ซึ่งมักจะดีกว่า Function Expression

เรียกว่า "ฟังก์ชันลูกศร" เพราะหน้าตาเป็นแบบนี้:

```js
let func = (arg1, arg2, ..., argN) => expression;
```

นี่จะสร้างฟังก์ชัน `func` ที่รับอาร์กิวเมนต์ `arg1..argN` จากนั้นประเมินค่า `expression` ทางด้านขวาด้วยอาร์กิวเมนต์เหล่านั้น แล้วคืนค่าผลลัพธ์

พูดง่ายๆ ก็คือ เป็นเวอร์ชันสั้นกว่าของ:

```js
let func = function(arg1, arg2, ..., argN) {
  return expression;
};
```

มาดูตัวอย่างที่เป็นรูปธรรมกัน:

```js run
let sum = (a, b) => a + b;

/* ฟังก์ชันลูกศรนี้เป็นรูปแบบสั้นกว่าของ:

let sum = function(a, b) {
  return a + b;
};
*/

alert( sum(1, 2) ); // 3
```

จะเห็นว่า `(a, b) => a + b` หมายถึงฟังก์ชันที่รับอาร์กิวเมนต์สองตัวชื่อ `a` และ `b` เมื่อรัน จะประเมินนิพจน์ `a + b` แล้วคืนค่าผลลัพธ์

- ถ้ามีอาร์กิวเมนต์เพียงตัวเดียว วงเล็บล้อมรอบพารามิเตอร์สามารถละได้ ทำให้สั้นลงไปอีก

    ยกตัวอย่างเช่น:

    ```js run
    *!*
    let double = n => n * 2;
    // คล้ายๆ กับ: let double = function(n) { return n * 2 }
    */!*

    alert( double(3) ); // 6
    ```

- ถ้าไม่มีอาร์กิวเมนต์เลย วงเล็บจะว่างเปล่า แต่ต้องใส่ไว้:

    ```js run
    let sayHi = () => alert("Hello!");

    sayHi();
    ```

ฟังก์ชันลูกศรสามารถใช้ได้ในลักษณะเดียวกับ Function Expression

ยกตัวอย่างเช่น สำหรับการสร้างฟังก์ชันแบบไดนามิก:

```js run
let age = prompt("What is your age?", 18);

let welcome = (age < 18) ?
  () => alert('Hello!') :
  () => alert("Greetings!");

welcome();
```

ฟังก์ชันลูกศรอาจดูไม่คุ้นตาและอ่านยากในตอนแรก แต่พอตาชินกับโครงสร้างแล้ว ก็จะเปลี่ยนไปอย่างรวดเร็ว

สะดวกมากสำหรับงานง่ายๆ บรรทัดเดียว ในยามที่ขี้เกียจเขียนโค้ดยาวๆ

## ฟังก์ชันลูกศรแบบหลายบรรทัด

ฟังก์ชันลูกศรที่เห็นมาจนถึงตอนนี้ค่อนข้างเรียบง่าย — รับอาร์กิวเมนต์จากด้านซ้ายของ `=>` ประเมินค่า แล้วคืนค่านิพจน์ทางขวา

บางครั้งต้องการฟังก์ชันที่ซับซ้อนกว่านั้น ที่มีหลายนิพจน์และคำสั่ง ในกรณีนี้ครอบด้วยวงเล็บปีกกาได้เลย ความแตกต่างหลักคือต้องมีคำสั่ง `return` ภายใน เพื่อคืนค่า (เหมือนฟังก์ชันปกติ)

ตัวอย่างเช่น:

```js run
let sum = (a, b) => {  // วงเล็บปีกกาเปิดฟังก์ชันแบบหลายบรรทัด
  let result = a + b;
*!*
  return result; // ถ้าเราใช้วงเล็บปีกกา ต้องใส่ "return" อย่างชัดเจน
*/!*
};

alert( sum(1, 2) ); // 3
```

```smart header="ยังมีอีกมากมาย"
สำหรับตอนนี้ เราสามารถใช้ฟังก์ชันลูกศรกับการทำงานแบบบรรทัดเดียวและฟังก์ชันคอลแบ็กได้แล้ว
```

## สรุป

ฟังก์ชันลูกศรใช้สะดวกสำหรับงานง่ายๆ โดยเฉพาะแบบบรรทัดเดียว มีสองรูปแบบ:

1. ไม่มีวงเล็บปีกกา: `(...args) => expression` -- ทางขวาเป็นนิพจน์: ฟังก์ชันจะประเมินค่าและคืนค่าผลลัพธ์ สามารถละวงเล็บได้ถ้ามีอาร์กิวเมนต์เพียงตัวเดียว เช่น `n => n*2`
2. มีวงเล็บปีกกา: `(...args) => { body }` -- วงเล็บปีกกาช่วยให้เขียนหลายคำสั่งภายในฟังก์ชันได้ แต่ต้องใส่ `return` อย่างชัดเจนเพื่อคืนค่าออกมา

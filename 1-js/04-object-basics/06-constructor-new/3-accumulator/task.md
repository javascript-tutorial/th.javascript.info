# สร้าง Accumulator ใหม่

ให้สร้างฟังก์ชัน constructor ชื่อ `Accumulator(startingValue)` ขึ้นมา

ออบเจ็กต์ที่สร้างขึ้นควรทำงานได้ดังนี้:

- เก็บ "ค่าปัจจุบัน" ไว้ในพร็อพเพอร์ตี้ `value` โดยค่าเริ่มต้นจะเท่ากับอาร์กิวเมนต์ `startingValue` ที่ส่งเข้ามาตอนสร้างออบเจ็กต์

- มีเมธอด `read()` ที่ใช้ `prompt` รับตัวเลขจากผู้ใช้ แล้วนำไปบวกเพิ่มเข้ากับ `value`

พูดง่ายๆ ก็คือ พร็อพเพอร์ตี้ `value` จะเก็บผลรวมของค่าเริ่มต้น `startingValue` กับตัวเลขทั้งหมดที่ผู้ใช้ป้อนเข้ามาผ่าน `read()`

ลองดูตัวอย่างการใช้งาน:

```js
let accumulator = new Accumulator(1); // ค่าเริ่มต้นเป็น 1

accumulator.read(); // รับค่าจากผู้ใช้แล้วบวกเข้ากับ value
accumulator.read(); // รับค่าจากผู้ใช้แล้วบวกเพิ่มอีก

alert(accumulator.value); // แสดงผลรวมทั้งหมด
```

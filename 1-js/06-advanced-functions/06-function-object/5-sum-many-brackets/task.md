# ผลรวมที่เรียกต่อกันกี่วงเล็บก็ได้

เขียนฟังก์ชัน `sum` ที่ทำงานได้แบบนี้:

```js
sum(1)(2) == 3; // 1 + 2
sum(1)(2)(3) == 6; // 1 + 2 + 3
sum(5)(-1)(2) == 6
sum(6)(-1)(-2)(-3) == 0
sum(0)(1)(2)(3)(4)(5) == 15
```

P.S. คำใบ้: อาจต้องกำหนดการแปลงออบเจ็กต์เป็นค่า primitive ให้ฟังก์ชันของคุณ

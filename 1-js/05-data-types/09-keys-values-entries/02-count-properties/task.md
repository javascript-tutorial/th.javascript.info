# นับพร็อพเพอร์ตี้

เขียนฟังก์ชัน `count(obj)` ที่คืนค่าจำนวนพร็อพเพอร์ตี้ในออบเจ็กต์:

```js
let user = {
  name: 'John',
  age: 30
};

alert( count(user) ); // 2
```

พยายามเขียนโค้ดให้สั้นที่สุด

หมายเหตุ: ไม่ต้องนับพร็อพเพอร์ตี้ที่เป็น Symbol นับเฉพาะพร็อพเพอร์ตี้ปกติเท่านั้น
